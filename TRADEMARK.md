# Trademark Policy

## Name

"Farol" is the name of this open-source project. The name is not registered as a trademark but we ask that it be used respectfully.

## Permitted Uses

- Referring to the farol project or software in documentation, articles, and talks.
- Using the name in package managers to distribute unmodified builds (e.g., `pip install farol`).
- Stating compatibility (e.g., "works with farol", "farol plugin for X").

## Restricted Uses

- Do not use "farol" in a way that implies official endorsement without permission.
- Do not use the name for derivative works that are substantially modified without clearly indicating they are forks (e.g., "farol-fork" or "myfork, based on farol").
- Do not use the project logo for commercial purposes without written permission.

## Logo

The farol logo, if/when created, will be licensed separately. Contact maintainers for usage outside of the permitted cases above.
