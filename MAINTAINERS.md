## Areas of Responsibility

- **core** — rendering engine, build pipeline, caching (`crates/farol-core/`)
- **cli** — command-line interface, dev server (`crates/farol-cli/`)
- **python** — PyO3 bindings, plugin host (`crates/farol-py/`, `python/`)
- **theme** — templates, assets, CSS/JS (`crates/farol-core/src/theme/`)
- **ci** — GitHub Actions, release pipeline (`.github/`)

## Becoming a Maintainer

As the project grows, contributors with sustained high-quality contributions may be invited to become maintainers for specific areas. See [GOVERNANCE.md](GOVERNANCE.md) for the decision model.
